Project 4

Next Wave Fest

Software

For this project, we had to design a website for a music festival called Next Wave Fest. The goal was to create a visually appealing and informative site that would attract young music lovers and showcase emerging artists. We used HTML and CSS to build the structure and style of the site. The webpage included sections for the festival introduction, lineup of artists, ticket information, and location details. We focused on making the design vibrant and modern to match the festival's energy. It was a good challenge to organize the content using flexbox and grid. Additionally, we used media queries to ensure the site was responsive on mobile devices. We also added a navigation menu to make it easy for visitors to explore different sections.

⟡ Reflection ⟡

1. What difficulties or obstacles did you experience while working on this project?

I struggled with making the design look modern and attractive while keeping the navigation simple. Also, making the website responsive was more complicated than I expected.

2. What skills did you apply during this project?

I applied my HTML and CSS skills to structure and style the webpage, and I used flexbox and grid to organize the layout. I also practiced using media queries to make the site responsive.

3. Did you develop or improve any new skills? If so, how?

Yes, I improved my ability to use media queries to adjust the layout for different screen sizes and learned how to create a more visually appealing design with CSS animations.

4. How could you improve this project if given more time or resources?

If I had more time, I would add more interactive elements like hover effects and smooth transitions to make the website more engaging and professional looking.
